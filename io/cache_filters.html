Using cache_filters (cf) moduleÂ¶

It lets you to download the filter data fromn SVO FPS and save it locally on disk as cache, so that you can access it faster. Hence you can even use wsynphot package offline as per your convinience. It also provides functions to load the required filter data from the cache instantaneously.

[1]:

import wsynphot.io.cache_filters as cf
